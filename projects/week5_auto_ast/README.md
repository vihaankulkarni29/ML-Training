# Week 5: Automated Antibiotic Susceptibility Testing (Auto AST)

Computer vision-based system for detecting and analyzing antibiotic resistance test plates.

## Project Structure

```
week5_auto_ast/
├── data/
│   └── test_plate.jpg    # Test plate image
├── src/
│   ├── detect_zones.py   # Computer Vision Logic
│   └── main.py           # Main entry point
├── requirements.txt
└── README.md
```

## Installation

Install required packages:
```bash
pip install -r requirements.txt
```

## Usage

Run the main program:
```bash
python src/main.py
```
